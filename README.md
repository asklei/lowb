# lowb

`lowb` 是一款使用NodeJs开发的命令行程序。

## 安装

**从npm下载安装**

```bash
npm install -g lowb
```

**源码安装**

```bash
git clone https://github.com/biezhi/lowb.git
cd lowb
npm install -g
```

## 使用

```bash
lowb --help

  Usage: lowb [options]


  Options:

    -V, --version    output the version number
    -i, --index <n>  ascii art index, default is random
    -h, --help       output usage information
```
